# EasyLot HUD 1.0 著陸頁 — 設計風格指南

## 選定設計方向：**高科技交易儀表板美學**

### 設計哲學
本著陸頁採用**專業深色主題**，呈現現代金融交易軟體的高端質感。設計靈感源自專業交易平台（如 Bloomberg Terminal、TradingView Pro）的視覺語言，強調**數據清晰度、即時反應、專業可信度**。

### 核心原則
1. **深色優先，翡翠綠與玫瑰紅點綴** — 深藍/深灰背景提升專業感，翡翠綠代表獲利/安全，玫瑰紅代表風險/止損
2. **極簡但不單調** — 避免過度裝飾，每個視覺元素都服務於信息傳達
3. **即時感與動感** — 細微的動畫與過渡效果暗示系統的實時性與反應靈敏度
4. **數據視覺化優先** — 圖表、線條、數字是視覺焦點，文案輔助

### 色彩哲學
- **主背景**：深灰藍 (`#0f1419` 或 `oklch(0.08 0.01 260)`) — 降低眼睛疲勞，突出前景元素
- **次背景**：稍淺深灰 (`#1a1f2e` 或 `oklch(0.12 0.01 260)`) — 卡片、容器
- **翡翠綠**：`#10b981` 或 `oklch(0.62 0.22 165)` — 止盈、獲利、正向指標
- **玫瑰紅**：`#f43f5e` 或 `oklch(0.58 0.25 15)` — 止損、風險、警告
- **文字**：淺灰/白 (`#e5e7eb` 或 `oklch(0.92 0.01 260)`) — 高對比度
- **強調色**：淺藍 (`#3b82f6` 或 `oklch(0.60 0.22 250)`) — CTA 按鈕、互動元素

### 佈局範式
- **非對稱非中心** — Hero 區採用左文案右圖片的**不對稱分割**，打破常見的居中單欄佈局
- **卡片網格** — 功能區使用 2x2 卡片網格，每卡片內部採用**圖示 + 標題 + 描述**的垂直堆疊
- **流動式分割線** — 使用 SVG 波浪分割線連接各區塊，暗示流動與連貫性
- **視覺層級** — 通過背景深度、陰影、尺寸差異建立清晰的視覺層級

### 簽名視覺元素
1. **翡翠綠/玫瑰紅線條** — 在卡片邊框、圖表線條、按鈕邊框中反覆出現
2. **微妙的玻璃態效果** — 卡片使用 `backdrop-blur` 與半透明背景，暗示現代感與深度
3. **數字與圖表** — 大型數字、K線圖、風險計量表等金融元素作為視覺裝飾

### 互動哲學
- **即時反饋** — 按鈕按下時有 `scale(0.97)` 的縮放反饋，150ms 內完成
- **懸停增強** — 卡片懸停時亮度提升、陰影加深，暗示可交互性
- **平滑過渡** — 所有狀態變化（顏色、尺寸、位置）都用 200-300ms 的 cubic-bezier 過渡
- **視覺引導** — 通過顏色漸變、箭頭、動畫引導用戶目光流向 CTA

### 動畫指南
- **進場動畫** — 卡片從下方淡入滑上，延遲 50-100ms 遞增，營造層疊感
- **懸停動畫** — 卡片邊框顏色漸變（翡翠綠 → 玫瑰紅），200ms 完成
- **按鈕動畫** — 點擊時 `scale(0.95)` + 顏色加深，160ms 彈性緩動
- **滾動觸發** — 使用 Intersection Observer 在元素進入視口時觸發進場動畫
- **禁用減速動畫** — 尊重 `prefers-reduced-motion`，關鍵交互仍保留但移除裝飾動畫

### 字體系統
- **顯示字體**：`Poppins` 或 `Sora` (700/800 weight) — 標題、大號文案，現代感強
- **正文字體**：`Inter` 或 `Outfit` (400/500 weight) — 段落、說明文字，高可讀性
- **等寬字體**：`JetBrains Mono` — 代碼片段、數字顯示（如風險倍數、手數）
- **字體層級**：
  - H1（Hero 標題）：Poppins 700, 48px
  - H2（區塊標題）：Poppins 600, 32px
  - H3（卡片標題）：Poppins 600, 20px
  - 正文：Inter 400, 16px
  - 說明文字：Inter 400, 14px, 灰色

### 品牌本質
**一句定位**：為專業交易者打造的即時風險管家，讓複雜的下單邏輯變得簡單直觀。

**三個性格形容詞**：
1. **專業** — 源自金融行業最佳實踐
2. **直觀** — 複雜功能簡化為一鍵操作
3. **可靠** — RSA 離線授權保障隱私與安全

### 品牌聲音
- **標題風格**：直接、有力、數據驅動。避免虛浮的營銷語言。
  - ✅ 「即時紅綠線盈虧預估」
  - ❌ 「歡迎來到我們的平台」
- **CTA 風格**：行動導向、明確價值。
  - ✅ 「申請 7 天試用」
  - ✅ 「立即體驗智慧下單」
  - ❌ 「點擊這裡」
- **微文案**：簡潔、技術性、充滿信心。
  - ✅ 「RSA 離線授權 — 零延遲，極致隱私」
  - ❌ 「我們很安全」

### Logo 與視覺標識
- **Logo 概念**：一個**綠色上升箭頭與紅色下降箭頭交叉**的幾何符號，中心是一個**圓形風險計量表**，代表盈虧與風險管理的核心。
- **Logo 樣式**：粗線條、現代幾何、無文字，可在深色背景上清晰顯示。
- **尺寸應用**：Header 中 32x32px，Favicon 中 16x16px。

### 簽名品牌色
**翡翠綠 `#10b981`** — 這是 EasyLot HUD 的專屬色，代表獲利、安全、正向操作。在整個頁面中作為主要強調色出現（按鈕、邊框、圖表線條）。
